## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-initialize`
- `allow-register-listener`
- `allow-remove-listener`
- `allow-set-source`
- `allow-play`
- `allow-pause`
- `allow-seek-to`
- `allow-set-rate`
- `allow-get-state`
- `allow-get-progress-checkpoint`
- `allow-clear-progress-checkpoint`
- `allow-dispose`
